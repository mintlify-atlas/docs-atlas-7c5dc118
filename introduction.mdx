---
title: "Introduction"
description: "OpenPanel is an open-source web and product analytics platform — a privacy-first alternative to Mixpanel, Google Analytics 4, and Plausible."
---

OpenPanel combines the depth of Mixpanel (funnels, cohorts, user profiles, session replay) with the simplicity and privacy focus of Plausible. It is fully open-source, self-hostable, and built for developers who want complete control over their analytics data.

## How OpenPanel compares

The table below shows how OpenPanel stacks up against other popular analytics platforms.

| Feature | OpenPanel | Mixpanel | GA4 | Plausible |
|---|---|---|---|---|
| Open-source | ✅ | ❌ | ❌ | ✅ |
| Self-hosting | ✅ | ❌ | ❌ | ✅ |
| Cookieless by default | ✅ | ❌ | ❌ | ✅ |
| Real-time dashboards | ✅ | ✅ | ❌ | ✅ |
| Funnels & cohort analysis | ✅ | ✅ | ✅* | ✅*** |
| User profiles & session history | ✅ | ✅ | ❌ | ❌ |
| Session replay | ✅ | ✅**** | ❌ | ❌ |
| Custom dashboards & charts | ✅ | ✅ | ✅ | ❌ |
| Event & funnel notifications | ✅ | ✅ | ❌ | ❌ |
| GDPR-compliant tracking | ✅ | ✅ | ❌** | ✅ |
| SDKs (Web, Swift, Kotlin, React Native) | ✅ | ✅ | ✅ | ❌ |
| Transparent pricing | ✅ | ❌ | ✅* | ✅ |
| Built for developers | ✅ | ✅ | ❌ | ✅ |
| A/B testing & variant breakdowns | ✅ | ✅ | ❌ | ❌ |

> \* GA4 has a free tier but often requires BigQuery (paid) for raw data access.
>
> \*\* GA4 has faced GDPR bans in several EU countries due to data transfers to US servers.
>
> \*\*\* Plausible supports simple goals only.
>
> \*\*\*\* Mixpanel session replay is capped at 5 000 sessions/month on the free plan and 20 000 on paid. OpenPanel has no limit.

## Tech stack

OpenPanel is built on a modern, proven open-source stack.

| Layer | Technology |
|---|---|
| Dashboard | Next.js |
| Event API | Fastify |
| Relational database | PostgreSQL |
| Event store | ClickHouse |
| Cache, pub/sub, queue | Redis + BullMQ |
| Proxy | Caddy |
| Auth | Oslo + Arctic |
| Internal API | tRPC |
| Styling | Tailwind CSS + shadcn/ui |
| Email | Resend |
